Learning Javascript Language

           




In the JavaScript language, describe all the use cases for each of:

parentheses (): - e.g. '1. Execute a function var result = doSomething()'

Parentheses are most commonly used when passing information into a function, or to provide information to loops (for, while) and conditionals (if, else if).

The JavaScript syntax dictates that a function definition must have a pair of parentheses even if the function doesn't actually need any arguments. Likewise, when calling a function (i.e. telling it to do what it's supposed to do), the (possibly empty) pair of parentheses is absolutely required.

Parentheses have multiple functions relating to functions and structures. They are used to contain a list of parameters passed to functions and control structures and they are used to group expressions to control the order of execution. Some functions have no parameters and in this case, the space between parentheses is blank. 


      Example: 
        var a = (4 + 3) * 2;       // Grouping expressions
            if (a > 10) {          // Containing expressions
              line(a, 0, a, 100);  // Containing a list of parameters
            }

brackets [ ]:

Square brackets are used to index (access) elements in arrays and also Strings. Specifically cells[i] will evaluate to the ith item in the array named cells.

Square Brackets [ ] are also used to access properties of Objects.


      Example:
      var myObject = {
        name: 'Prem'
        favorite_food: 'Butter Chicken'
      }
 
      console.log(myObject['name']) // Logs 'Prem'
 
      console.log(myObject.name') // Simpler form of above, also logs 'Prem'

braces { }:

The braces or curly brackets are used to define the start and end of the function, they also separate code into blocks within the function. The braces help JavaScript to understand the structure of our script and what we want it to do.


      Example:
      function MyFunction() {
      alert("Hello world!")
      }

It is also used to separate blocks of code within the function, like this:

 
      function MyFunction() {
      if (a===1) {
      alert("Hello world!")
      } else {
      alert("Goodbye world!")
      }
      }

This means that if the variable "a" is equal to 1 the message "Hello world!" is displayed, else the message "Goodbye world!" is displayed.

single quotes ' ' & double quotes " ":

JavaScript, allows you to use either double quote or single quote marks to create a string.


      Example:

      var x ="Hello World!"; 
      var x = ‘Hello World!’;

Mixing up the single and double quote in same string is not allowed.


      Example:

      var x = ‘Hello World!";

Quote mark within a quote:

The use of quotations within a string generates error.


      Example:

      var message=’It’s alright’;

There is no way for the browser to know which one is the closing quote. The interpreter sees the second quote in 'Javascript's as the ending quote - so rest of the line becomes invalid.

To escape a character you precede it with forward slash mark \.


      Example:

      var message=’It\’s alright’;
