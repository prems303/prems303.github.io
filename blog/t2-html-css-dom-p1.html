Can you think of an anology to describe HTML, CSS and DOM?

HTML (Hyper Text Markup Language) is the standard markup language to create web pages along with CSS (Casacading Style Sheets) which adds styles and DOM (Document Object Model) is described as a tree structure which means the nodes of all the documents are organised.

Basically HTML is like the structure of the house. CSS represents colours, designs and styles in the house. DOM tells us about the number of contents of the house eg: bedrooms, bathrooms, kitchen and garage etc.

What is meant by boxifying design?

Before developing the websites - a plan should be made regarfing the placing of images and text in an appropiate manner so that it looks pleasing and attractive to the visitors of the page/site. Boxifying helps us to divide, design, place & organise all the elements in the page.

What is the box model?

All the HTML elements contains boxes. So when considering about designing & layout, the CSS Box Model is used where a box consists of the content, padding, borders and margins.
