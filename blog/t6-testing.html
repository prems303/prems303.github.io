Scope

           






What 'scope' is, and how it works in JavaScript.

In Javascript, scope is the set of variables, objects and functions you have access to. There are two types of variables – Global variables and Local Variables. A global variable is accessible from anywhere in the code whereas a local variable can be accessed only with the declared function.

Example of Global Variable

 
      var a = 1;
       
      function one() {
        console.log(a);  //returns ‘1’
      }

Example of Local Variable


      function two() {
        var a = 2;
          console.log(a);  //returns '2' 
      }

Another example of Local Variable


      var a = 1;
 
      function two() {
        var a = 2;
          console.log(a); 
      }
      //returns 2 instead of 1 as ‘a’ is declared again in a function 
      //which will overwrite the globally declared value.
