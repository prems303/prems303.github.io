What is grid based design?

Grids offer an effective design approach for website layouts. One has to study the grid and understand how to successfully use it to create the site layouts. Now, most of the sites are created by using the 12 column structural layout. By dividing the row into 12 columns for organising and presenting the information.

Whats all hype about responsive webpages?

Responsive web pages facilitates us to view the web page according to the user requirements. As the technology is progressing more & more devices of different viewing dimensions are being released, so this can be developed by using @media query.

What is semantic structure?

A semantic element clearly describes its meaning to both the browser and the developer.

  • Examples of semantic elements: < header >, < form >, < table > and < img > - Clearly defines its content.
  • Examples of non-semantic elements: < div > and < span > - Tells nothing about its content.
